# dircnt
The most basic UNIX C program for counting files and their total size in a directory.

Usage:
```
dircnt [dir]
```
Useful when running with the `watch` utility, for arbitrary monitoring purposes:
```
watch [-n interval_seconds] 'dircnt [dir]'
```
